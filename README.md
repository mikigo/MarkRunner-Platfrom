# MarkRunner-Platfrom

### 本地启动

#### 后端

#### 使用 Pip 安装依赖

1. 安装依赖
```sh
pip install -r requirements.txt
```

2. 启动服务
```sh
python run.py
```

服务现在应该正在运行，访问 http://localhost:9999/docs 查看API文档

#### 前端

启动项目需要以下环境：
- node v18.8.0+

1. 进入前端目录
```sh
cd web
```

2. 安装依赖
```sh
npm i
```

3. 启动
```sh
pnpm dev
```
